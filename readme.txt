== Description ==

Control XMLRPC, auto updates, email sending, and debug mode in WordPress with FwP Essentials.


### Features

* Disable XMLRPC protocol to make WordPress more secure from SSRF attacks.
* Control auto updates for core WordPress, themes, and plugins.
* Disable email sending in WordPress.
* Enable debug mode in WordPress.



= Installation Instructions =
- Download the plugin from https://fahhad.io/fwp-essentials-plugin/
- In your WordPress dashboard, go to Plugins > Add New
- Click on the "Upload Plugin" button and select the fwp-essentials .zip file
- Click on the "Install Now" button
- Once installed, click on the "Activate Plugin" button
- Go to the FwP Essentials menu in your WordPress dashboard to access the plugin settings
